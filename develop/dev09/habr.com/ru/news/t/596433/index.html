Microsoft заблокировала обход браузера по умолчанию в Windows 10 и 11

  • Браузеры,
  • Microsoft Edge,
  • Разработка под Windows

В декабрьском обновлении Windows 10 и 11 Microsoft предприняла шаги для блокировки использования сторонних браузеров, чтобы пользователи больше обращались к ее собственному браузеру Edge.

В обновлениях KB5008212 и KB5008215 компания реализовала блокировку обходных путей, используемых EdgeDeflector и Firefox для открытия ссылок в браузере, отличном от Edge. Microsoft использовала протокол microsoft-edge: // для принудительного открытия ссылок, например, в виджетах Windows 11.

Инструмент EdgeDeflector мог перехватывать такие ссылки и перенаправлять их для открытия в выбранном пользователем браузере. Однако обновление заблокировало его. То же и с Firefox.

В настоящее время пользователям еще доступны другие аналогичные инструменты, например, MSEdgeRedirect. Он имеет функции, подобные EdgeDeflector.

Однако не исключено, что Microsoft заблокирует использование всех таких инструментов уже в ближайшем будущем.

В начале декабря стало известно, что Microsoft начала настойчиво с помощью уведомления предупреждать пользователей Windows 10 и 11, что Edge лучше Google Chrome. Предупреждения появляются тогда, когда пользователь пытается скачать Chrome из Edge.

15 января 2020 года Microsoft выпустила релизную версию браузера Edge на движке Chromium. Microsoft заявила, что новый Edge «является единственным браузером, оптимизированным для Windows 10». Компания опубликовала исходный код некоторых компонентов браузера (в фильтре набрать: microsoft edge).

В начале февраля 2020 года Microsoft стала агрессивно рекламировать свой новый браузер Edge в разделе с рекомендациями в меню «Пуск» Windows 10. Это происходило, например, если у пользователя установлен браузер Firefox по умолчанию. В этом случае рекламный баннер компании гласит: «Ещё пользуетесь Firefox? Microsoft Edge здесь». Кликая по нему, пользователь начинает загрузку нового браузера Microsoft Edge на движке Chromium.
